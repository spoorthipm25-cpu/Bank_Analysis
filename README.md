# 🛒 Supermarket Transaction Data Analysis

## 📊 Project Overview
This project focuses on analyzing supermarket transaction data to uncover meaningful insights about customer behavior, sales trends, and product performance. The dataset was cleaned, transformed, and analyzed using Microsoft Excel.

---

## 🧾 Dataset Details
- Total Records: ~50,000+ transactions
- Data Source: Provided dataset
- File Formats: Excel (.xlsx) and CSV (.csv)

### Key Columns:
- Customer ID
- Product Details
- Transaction Date & Time
- Sales Amount
- Customer Type (Member, Gold, Corporate, Premium, Non-member)

---

## 🔧 Data Cleaning & Preparation
- Removed unnecessary columns (e.g., unnamed columns)
- Handled missing/null values
- Standardized data formats
- Organized data into a structured format
- Created a new analysis sheet for reporting

---

## 📈 Analysis Performed
- Sales distribution across different customer types
- Transaction trends over time
- Identification of high-value customers
- Product/category performance analysis
- Data summarization using Excel tools

---

## 📊 Key Insights
- Certain customer segments (e.g., Gold/Premium) contribute higher revenue
- Repeat customers play a major role in total sales
- Sales patterns vary based on time and transaction frequency
- Data cleaning significantly improved analysis accuracy

  - Data Cleaning
